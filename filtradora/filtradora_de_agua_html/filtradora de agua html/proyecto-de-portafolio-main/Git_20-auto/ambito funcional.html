Proyecto filtrado de agua

¿Qué se entiende por Ámbito Funcional?

El Ámbito Funcional de un proyecto de filtrado de agua comprende todas las acciones y beneficios relacionados con la purificación del recurso hídrico. Este sistema se encarga de eliminar impurezas físicas como arena, tierra y sedimentos; reducir contaminantes químicos como cloro, metales pesados o sales en exceso; y disminuir la presencia de microorganismos que pueden causar enfermedades. Su funcionalidad no se limita únicamente al consumo humano, sino que también se extiende a la industria, la agricultura y los espacios educativos, donde el agua de calidad es indispensable. Además, un proyecto de filtración contribuye a la sostenibilidad ambiental al promover el uso responsable del recurso y disminuir la dependencia de agua embotellada, generando así un impacto positivo en la salud pública, la economía y el medio ambiente.

Diseño y Desarrollo Mecatrónico

Diseño mecánico:

  • Se construye la estructura de la filtradora utilizando materiales resistentes como acrílico, PVC y soportes metálicos.
  • Se organizan los compartimentos donde se ubican los filtros físicos y químicos para garantizar un flujo de agua continuo.

    • Diseño electrónico:

  • Implementación de un Arduino UNO como unidad de control.
  • Uso de sensores de nivel y calidad de agua (por ejemplo, turbidez o pH) para monitorear el estado del recurso hídrico.
  • Incorporación de bombas de agua y válvulas eléctricas controladas por relés para dirigir el flujo de agua dentro del sistema de filtrado.

    • Interacción del sistema:

  • Cada componente (sensor, bomba, válvula, módulo Bluetooth) trabaja en conjunto como un sistema integrado, lo que refleja la esencia de la mecatrónica: combinar mecánica + electrónica + programación para ofrecer una solución funcional.

    • Punto Clave

    El punto clave del proyecto de la filtradora de agua es la automatización y el control inteligente del proceso de purificación. A través de sensores conectados al Arduino UNO, se monitorea en tiempo real la calidad del agua, mientras que el sistema activa automáticamente bombas y válvulas según sea necesario. Además, el módulo Bluetooth permite supervisar y controlar el proceso desde una aplicación móvil, garantizando un uso eficiente del recurso hídrico. En conclusión, el proyecto combina mecatrónica y sostenibilidad, ofreciendo un sistema confiable, automatizado y accesible para mejorar la calidad del agua.

    Énfasis en Mecatrónica

    Este proyecto integra la mecánica, la electrónica y la programación en un sistema de control inteligente para la filtradora, logrando una solución innovadora, sostenible y educativa.
